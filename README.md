# java-cli-maven-janusgraph-cassandra-simple

## Description
A POC for janusgraph
connecting to a database.

## Tech stack
- java
- maven
  - janusgraph
  - cassandradb connector

## Docker stack
- janusgraph
- cassandra
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
